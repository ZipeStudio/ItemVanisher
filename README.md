# Description
Client Fabric Mod that allows you to disable/enable render of items on the ground at the toggle of a button.

Disables rendering of items on the ground:
- Item Models
- Item HitBoxes
- Item Shadows

| Keybinding | Default key|
| ----------- | ---------- |
| Toggle Rendering | i |
